# InternetCommunication
UDP and FTP cpp code

FileToFtp
功能：将本地文件或文件夹中所有文件上传到指定FTP服务器上
环境：Ubuntu+Clion

FileToFtp_win
功能：将本地文件或文件夹中所有文件上传到指定FTP服务器上
环境：Windows+Visual Studio 2019 win32

UdpSned_Ubuntu
功能：将字符串以UDP协议循环发送
环境：Ubuntu+Clion

UdpSned_windows
功能：将字符串以UDP协议循环发送
环境：Windows+Visual Studio 2019 x64

UdpReceive_windows
功能：以UDP协议循环接收字符串
环境：Windows+Visual Studio 2019 x64
